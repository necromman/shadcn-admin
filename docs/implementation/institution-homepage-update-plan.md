# 기관별 홈페이지 업데이트 계획

## 📋 요구사항 분석

### 1. 현재 문제점
- 기존 더미 데이터가 일반 공작소/제작소 형태로 되어 있음 (3D 프린터, 레이저 커터 등)
- 실제 장비 목록은 반도체/나노 전문 장비들 (전자현미경, CVD, 스퍼터 등)
- 메인 화면에 기관이 3개만 표시되는 한계
- 최근 추가된 기관 강조 필요

### 2. 개선 방향
- 더미 데이터를 반도체/나노 관련 전문 기관으로 변경
- 더 많은 기관 추가 (10개 이상)
- 메인 화면에 스크롤 가능한 캐러셀 형태로 변경
- 최근 추가된 기관 하이라이트 카드 추가

## 🏢 새로운 기관 컨셉

### 반도체/나노 전문 기관들
1. **한국나노융합기술원** - 나노 기술 전문
2. **서울반도체연구센터** - 반도체 공정 장비
3. **대전나노팹센터** - 나노 소자 제작
4. **경기반도체지원센터** - 반도체 테스트/분석
5. **부산전자소재연구소** - 전자 소재 분석
6. **광주포토닉스센터** - 광학/포토닉스
7. **인천마이크로시스템** - MEMS/마이크로시스템
8. **충북바이오칩센터** - 바이오칩/센서
9. **전북탄소융합기술원** - 탄소 소재
10. **울산화학소재연구소** - 화학 소재 분석
11. **강원신소재연구센터** - 신소재 개발
12. **제주청정에너지연구원** - 에너지 소재

## 🔧 장비 카테고리 재구성

### 주요 장비 분류
1. **분석 장비**
   - 전자현미경 (SEM, TEM)
   - X선 회절분석기 (XRD)
   - 질량분석기 (SIMS)
   - 원자현미경 (AFM)

2. **증착 장비**
   - 스퍼터 (Sputter)
   - 화학기상증착 (CVD)
   - 원자층증착 (ALD)
   - 전자빔증착 (E-beam)

3. **식각 장비**
   - 건식 식각 (Dry Etcher)
   - 습식 식각 (Wet Station)
   - 플라즈마 식각 (ICP)

4. **리소그래피 장비**
   - 노광기 (Stepper)
   - 전자빔 리소그래피 (EBL)
   - 마스크 얼라이너

5. **측정 장비**
   - 박막두께측정기
   - 전기특성측정기
   - 광학현미경

## 🎨 UI/UX 개선사항

### 메인 홈페이지 개선
1. **기관 섹션 리디자인**
   - 캐러셀/슬라이더 형태로 변경
   - 한 번에 3-4개씩 표시
   - 좌우 스크롤 또는 자동 슬라이드
   - 전체 보기 버튼 유지

2. **최근 추가 기관 섹션**
   - 상단에 별도 섹션 추가
   - "NEW" 뱃지 표시
   - 눈에 띄는 카드 디자인
   - 최대 3개까지 표시

3. **기관 카드 정보 강화**
   - 전문 분야 표시
   - 주요 장비 카테고리 아이콘
   - 위치 정보 추가

### 기관별 홈페이지 개선
1. **장비 섹션 전문화**
   - 카테고리별 탭 구성
   - 장비 스펙 상세 표시
   - 예약 가능 시간 표시

2. **서비스 내용 전문화**
   - 분석 서비스
   - 공정 서비스
   - 컨설팅 서비스
   - 교육 프로그램

## 📝 구현 순서

### Phase 1: 데이터 업데이트
1. ✅ 기존 더미 데이터 백업
2. ⬜ 새로운 전문 기관 데이터 생성 (12개)
3. ⬜ 실제 장비 목록 기반 데이터 구성
4. ⬜ 서비스 내용 전문화

### Phase 2: 메인 홈페이지 UI 개선
1. ⬜ 기관 섹션을 캐러셀로 변경
2. ⬜ 최근 추가 기관 섹션 추가
3. ⬜ 반응형 디자인 최적화

### Phase 3: 기관별 페이지 개선
1. ⬜ 장비 카테고리 탭 추가
2. ⬜ 장비 상세 정보 강화
3. ⬜ 전문 서비스 내용 추가

### Phase 4: 최종 검증
1. ⬜ 전체 기능 테스트
2. ⬜ 반응형 디자인 검증
3. ⬜ 빌드 및 타입 체크

## 🚀 예상 결과

### 개선된 사용자 경험
- 전문 기관의 실제 서비스를 반영한 현실적인 프로토타입
- 많은 기관을 효율적으로 탐색 가능
- 최신 기관 정보 쉽게 확인
- 전문 장비와 서비스 정보 명확히 전달

### 기술적 개선
- 확장 가능한 데이터 구조
- 효율적인 UI 컴포넌트
- 성능 최적화된 캐러셀
