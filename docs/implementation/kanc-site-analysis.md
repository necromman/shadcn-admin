# KANC 사이트 분석 및 프로토타입 설계 문서

## 📊 사이트 개요

한국나노기술원(KANC) 웹사이트는 두 개의 메인 포털로 구성:
1. **KANC 소개 홈페이지** (https://www.kanc.re.kr/main.do)
2. **KANC 서비스 홈페이지** (https://css.kanc.re.kr/cssMain.do)

## 🏗️ 네비게이션 구조

### 상단 네비게이션 (GNB)
```
├── KANC 소개 홈페이지 | KANC 서비스 홈페이지 (탭 전환)
├── Intranet | 홈페이지 안내 | Eng | SNS
└── 로고 영역
```

### 메인 메뉴 구조

#### KANC 소개 홈페이지 메뉴
1. **기술원 소개**
   - 인사말
   - 일반현황 (설립목적, 비전, 시설현황)
   - 주요사업 및 성과
   - CI소개
   - 인재상
   - 윤리헌장
   - 안전보건경영
   - 조직 및 직원
   - 찾아오시는길

2. **기술소개**
   - 공정기술 (모듈공정, 박막, 식각, 패터닝, 후공정)
   - 에피기술
   - 시험분석기술
   - 소자기술

3. **서비스 안내**
   - 팹서비스
   - 장비안내
   - 교육서비스
   - 임대서비스

4. **공지 및 안내**
   - 공지사항
   - 공고 (입찰, 채용)
   - 법령자료

5. **기업지원 안내**
   - 기업지원 현황
   - 기업지원 소개
   - 행사 안내
   - 기술 이전
   - 기술 사업화

6. **KANC 활동**
   - KANC소식
   - 관련 행사 안내
   - 이용자 협의회
   - 환경안전

7. **정보공개**
   - 정보공개제도안내
   - 사업실명제
   - 사전정보공개
   - 공공데이터 개방
   - 경영공시

#### KANC 서비스 홈페이지 메뉴
1. **시험분석서비스**
2. **교육서비스**
3. **팹서비스 종합안내**
4. **온라인 예약**
5. **고객서비스**

## 🎨 메인 페이지 컨텐츠 구조

### KANC 소개 홈페이지
1. **메인 비주얼 슬라이더**
   - 풀스크린 이미지 슬라이더
   - 타이틀, 서브타이틀, CTA 버튼

2. **퀵메뉴 섹션**
   - 6개 주요 서비스 아이콘 메뉴
   - 호버 효과

3. **공지사항 / 뉴스 탭**
   - 탭 전환 UI
   - 리스트 형태
   - 더보기 버튼

4. **배너 영역**
   - 관련 사이트 배너
   - 슬라이더 형태

5. **푸터**
   - 사이트맵
   - 연락처 정보
   - 관련 사이트 링크
   - 개인정보처리방침

### KANC 서비스 홈페이지
1. **메인 비주얼**
   - 서비스 중심 이미지
   - 예약 바로가기 버튼

2. **서비스 카드 섹션**
   - 4개 주요 서비스
   - 카드 레이아웃

3. **공지사항 / 뉴스**
   - 탭 UI
   - 최신 5개 항목

4. **예약 현황**
   - 달력 위젯
   - 빠른 예약

5. **고객지원**
   - FAQ
   - 문의하기

├── /services/* (서비스 안내)
