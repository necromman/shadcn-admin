## 🔄 페이지 플로우

### 신규 사용자
1. 메인 → 회원가입 → 본인인증 → 정보입력 → 가입완료
2. 로그인 → 메인 → 자료검색 → 도서상세 → 대출신청

### 기존 사용자
1. 로그인 → My Library → 대출현황 확인
2. 메인 → 좌석예약 → 좌석선택 → 예약완료
3. 자료검색 → 도서상세 → 예약 → My Library

### 관리 플로우
1. My Library → 대출도서 → 연장신청
2. My Library → 시설예약 → 예약취소
3. 서비스 → 희망도서 → 신청 → 신청내역
